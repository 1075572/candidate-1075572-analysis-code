# candidate-1075572-analysis-code
# Dynamic mechanical signal propagation through gelatine hydrogels

MBiol Part II research project, University of Oxford
Candidate number 1075572

## About

This repository holds the statistical analysis and figure generation code for an MBiol Part II research project. The project investigated whether fast mechanical signals propagate through gelatine hydrogels, using laser Doppler vibrometry to record surface velocity at nine distances from a stinger source, in response to two stimuli (a click and a frequency sweep).

## What this code does

The code in this repository covers two things.

- **Statistical analysis.** Friedman tests on peak surface velocity in the time domain, and Skillings-Mack tests on spectral peak amplitudes in the frequency domain.
- **Figure generation.** Time-domain traces, frequency spectra, and the distance-attenuation plots used in the dissertation and viva presentation.

Raw data acquisition and stimulus generation were performed in a custom MATLAB application (not included here).

Claude, an AI assistant developed by Anthropic, was used to support the writing of code in this repository. Tasks included code drafting, debugging, and refining figure layouts. All scientific decisions, including experimental design, data interpretation, and analysis choices, were made by the author.
